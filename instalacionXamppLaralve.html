1
Requisitos previos Obligatorio ▼
Sistema operativo compatible
✓ Windows 10 / 11 (64-bit)
✓ 4 GB RAM mínimo (8 GB recomendado)
Lo que instalaremos en esta guía
XAMPP (Apache + MySQL + PHP)
Composer (gestor de paquetes PHP)
Node.js + NPM
Laravel 11 (vía Composer)
2
Instalar XAMPP Apache + MySQL ▼
Paso 1 — Descargar XAMPP
Ir a apachefriends.org y descargar XAMPP para Windows (PHP 8.x)
Elige la versión con PHP 8.1+ para compatibilidad con Laravel 11
Paso 2 — Ejecutar el instalador
Ejecutar xampp-windows-x64-*.exe como administrador
Seleccionar componentes: Apache, MySQL, PHP, phpMyAdmin
Ruta de instalación: C:\xampp (no cambiar)
Si el antivirus bloquea la instalación, desactivarlo temporalmente o agregar excepción para la carpeta C:\xampp
Paso 3 — Iniciar servicios en XAMPP Control Panel
Abrir XAMPP Control Panel
Clic en Start junto a Apache → debe ponerse verde
Clic en Start junto a MySQL → debe ponerse verde
Verificar en el navegador
http://localhost
Debe aparecer el dashboard de XAMPP con fondo naranja
3
Verificar PHP y agregar al PATH + Instalar Composer PHP 8.x ▼
Verificar PHP en CMD
$php -v
Si dice "no reconocido", necesitas agregar PHP al PATH del sistema
Agregar PHP al PATH (solo si fue necesario)
Buscar "Variables de entorno" en el menú inicio de Windows
En "Variables del sistema" → seleccionar Path → Editar
Agregar nueva entrada: C:\xampp\php
Aceptar todo y reiniciar CMD
Instalar Composer
Descargar Composer-Setup.exe desde getcomposer.org
En el instalador, apuntar al PHP de XAMPP: C:\xampp\php\php.exe
Verificar instalación:
$composer --version # Esperado: Composer version 2.x.x
4
Instalar Node.js y NPM Frontend assets ▼
Descargar Node.js LTS
Ir a nodejs.org y descargar la versión LTS (20.x o 22.x)
Instalar con opciones por defecto (incluye NPM automáticamente)
Verificar en CMD
$node -v # v20.x.x o superior $npm -v # 10.x.x o superior
5
Instalar Laravel globalmente Laravel 11 ▼
Instalar Laravel Installer via Composer
$composer global require laravel/installer
Esto instala el comando laravel disponible en todo el sistema
Agregar Composer bin al PATH (si laravel no es reconocido)
Agregar al PATH: C:\Users\TuUsuario\AppData\Roaming\Composer\vendor\bin
Reemplaza "TuUsuario" con tu nombre de usuario real de Windows
Verificar Laravel Installer
$laravel --version # Laravel Installer 5.x.x
6
Crear y configurar un proyecto nuevo Nuevo proyecto ▼
Crear proyecto en htdocs de XAMPP
$cd C:\xampp\htdocs $laravel new mi-proyecto
El instalador preguntará qué starter kit quieres. Para empezar elige No starter kit
Levantar el servidor de desarrollo
$cd mi-proyecto $php artisan serve
Verificar en el navegador
http://127.0.0.1:8000
Debe aparecer la pantalla de bienvenida de Laravel
Configurar base de datos en .env
# Archivo: .env (raíz del proyecto) DB_CONNECTION=mysql DB_HOST=127.0.0.1 DB_PORT=3306 DB_DATABASE=mi_proyecto DB_USERNAME=root DB_PASSWORD= # En XAMPP, root no tiene contraseña por defecto
Crear la BD mi_proyecto en http://localhost/phpmyadmin
Ejecutar migraciones
$php artisan migrate
Crea las tablas por defecto (users, sessions, etc.) en la base de datos
7
Verificación final completa Checklist ▼
Comandos de verificación rápida
$php -v # PHP 8.x.x $composer --version # Composer 2.x.x $node -v # v20.x.x $npm -v # 10.x.x $laravel --version # Laravel Installer 5.x.x
URLs de verificación en navegador
http://localhost
Dashboard XAMPP (fondo naranja)
http://localhost/phpmyadmin
Panel de gestión MySQL
http://127.0.0.1:8000
Pantalla de bienvenida Laravel
Checklist final
Apache corre en XAMPP (puerto 80, luz verde)
MySQL corre en XAMPP (puerto 3306, luz verde)
php -v devuelve 8.x
composer --version devuelve 2.x
node -v y npm -v responden correctamente
Proyecto Laravel creado en C:\xampp\htdocs\mi-proyecto
php artisan serve levanta en el puerto 8000
php artisan migrate ejecuta sin errores
Pantalla de bienvenida Laravel visible en el navegador
